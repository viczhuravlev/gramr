# Getting Started

## Theme colors

<div
  style={{
    display: 'flex',
    justifyContent: 'space-between'
  }}
>
  {Object.keys(themeColors).map(color => (
    <div
      style={{
        display: 'inline-block',
        width: '75px',
        height: '75px',
        borderRadius: '50%',
        backgroundColor: themeColors[color],
        color: color === 'white' ? '#000' : '#fff',
        textAlign: 'center',
        lineHeight: '75px',
        fontSize: '14px',
        border: `1px solid ${color === 'white' ? '#000' : themeColors[color]}`
      }}
    >
      {themeColors[color]}
    </div>
  ))}
</div>
